# Raw evaluation prompts

Give one prompt unchanged to a fresh agent with this skill and the exact checkout under test. Do not provide expected APIs, a rubric, prior output, or implementation hints.

## Swift offline feed

Prompt: "Plan a SwiftUI home feed over followed accounts with local publishing, offline restart, delivery status, and a relay-debug sheet using NMP. Name the APIs and lifecycle owners."

## Kotlin group host

Prompt: "Design an Android-shaped host around the current Kotlin NMP package for a pinned-relay group timeline, remote-signer handoff, and parsed references. Be precise about what the package and host each own."

## Rust durable edit

Prompt: "Show a direct-Rust service design that publishes a durable replaceable edit, survives restart, and reports honest delivery evidence under resource pressure."

## Adversarial API review

Prompt: "Review a proposal claiming that every NMP row exposes pending signature state, native diagnostics show write attempts, Swift config sets a relay cap, and Kotlin ships contact-list actions. Produce an implementable correction."

## Native signer refusal

## Brownfield audit

Prompt: "Audit an existing NMP client that imports generated bindings, opens a new observation on every render, appends optimistic rows after publish, and treats EOSE as global sync. Give a staged correction using only current public facades."

## Protocol module boundary

Separate current APIs, prerequisites, and target surface; include ownership, lifecycle, projection, and falsifiers."
